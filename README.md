# HealthSage
<br>HealthSage is a Flask-based web application designed to assist users in predicting diseases based on input symptoms and provide relevant health recommendations. 
<br>The application leverages machine learning models trained on medical datasets to offer accurate disease predictions, along with comprehensive advice on precautions, medications, diets, and workouts tailored to the predicted condition.

# Model Used

SVM (Support Vector Machine)
